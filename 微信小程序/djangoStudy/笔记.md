## 通过命令行创建 django 程序
### 创建项目
* django-admin startproject 项目名称

### 项目关系
* manage.py 管理文件
* s28 项目目录
* * __init__.py 配置
* * settings.py 配置
* * urls.py 路由 --> url跟函数对应的关系
* * wsgi.py runserer命令就使用wsgiref 模块做简单的web server

### 运行django 项目
* 命令： python manage.py runserver [端口]/[0.0.0.1:80]
